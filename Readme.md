#  Heart Disease Prediction App (ML Project)

This project is a Machine Learning-based web application that predicts the likelihood of heart disease using patient health data. The model was developed after performing detailed Exploratory Data Analysis (EDA), data preprocessing, and feature engineering on a real-world dataset.

The application is built using Streamlit and allows users to input medical parameters to receive instant predictions along with risk assessment.

##  Key Highlights

* Performed **Exploratory Data Analysis (EDA)** to understand feature relationships and trends
* Applied **data preprocessing techniques** including:

  * Handling missing values
  * Feature encoding (one-hot encoding)
  * Feature scaling using StandardScaler
* Trained a **K-Nearest Neighbors (KNN)** model for classification
* Built an **interactive web interface** using Streamlit
* Integrated trained model using **Joblib for deployment**

##  Machine Learning Pipeline

1. Data Cleaning & Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Engineering & Encoding
4. Model Training (KNN)
5. Model Evaluation
6. Deployment using Streamlit

##  Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit
* Joblib
